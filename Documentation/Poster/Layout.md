# Tentative A0 Poster Layout

**Dimensions:** A0 (841mm x 1189mm) - Landscape or Portrait (Assuming Portrait for standard academic flow, but Landscape works for timeline/flow). *Recommendation: Landscape (1189mm wide x 841mm high) allows for a wider central diagram.*

### Style Guide
*   **Background:** White (A0 Landscape: 1189mm Ã— 841mm)
*   **Text:** Charcoal (#1D1D1D) - Georgia font (serif), clean, reading size 24-32pt
*   **Headings:** Gold (#C9AC78) - Georgia font (serif), 48-60pt, bold
*   **Primary Accents:** Gold (#C9AC78) for main features
*   **Secondary Accents:** Success Green (#5A9E6F) for positive metrics
*   **Alert Accents:** Danger Red (#C75050) for safety features
*   **Tech Accents:** Deep Blue (#4B4E9E) for AI models
*   **Cloud Accents:** Teal (#009688) for cloud services
*   **Style:** Flat, scientific, professional - suitable for academic poster
